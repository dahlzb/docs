#### 开机启动

``` shell
# 开机启动
docker update --restart=always 容器ID
# 取消
docker update --restart=no 容器ID 
```

#### 防火墙

```shell
# 开放防火墙指定端口
firewall-cmd --zone=public --add-port=6379/tcp --permanent
# 重新加载防火墙
firewall-cmd --reload
```
